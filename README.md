# AniDesk Extension Repository

Official extension repository for AniDesk desktop app.

## How to add this repo to AniDesk

1. Open AniDesk
2. Go to **Extensions** in the sidebar
3. Click the **Repos** tab
4. Click **Add**
5. Paste this URL:
   ```
   https://YOUR-GITHUB-USERNAME.github.io/anidesk-repo/index.json
   ```
6. Click **Add**
7. Go to **Browse** tab — all extensions will appear

## How this repo was built

This repo follows the AniDesk extension format:
- `index.json` — lists all extensions
- `extensions/*.js` — executable JS for each source
